## Example Templates

### Code Generation (GPT-5.2 / Claude 4.5)

```
You are an expert software developer. Generate clean, production-ready code.

Task: {task_description}
Language: {programming_language}
Requirements:
- Follow best practices
- Include error handling
- Add inline comments
- Optimize for performance

Output format: Complete code with explanation.
```

### Content Writing (Gemini)

```
Create engaging content for {platform}.

Topic: {topic}
Tone: {professional/casual/technical}
Length: {word_count} words
Target audience: {audience_description}

Include:
- Compelling headline
- Clear structure
- Call to action
```

### Data Annotation (Claude 4.5)

```
Annotate the following data for machine learning training.

Data type: {text/image_description/audio_transcript}
Task: {classification/entity_extraction/sentiment}
Labels: {list_of_labels}

Data:
{input_data}

Output format: JSON with annotations and confidence scores.
```

### Research Summarization (GPT-5.2)

```
Summarize the following research material.

Source: {source_type}
Focus areas: {key_topics}
Output length: {brief/detailed}

Content:
{research_content}

Provide:
- Key findings
- Methodology overview
- Implications
- Limitations
```
